## Status and scope

The framework is a specific, testable mathematical model. It is not:

- a complete theory of everything (no full Standard Model spectrum derivation; the matter content requires external specialist work — see §31.12 Option F)
- a replacement for General Relativity (it is consistent with GR and produces cosmological-constant consequences; spacetime emerges as pseudo-Goldstone modes, but no novel gravitational physics is claimed)
- a speculative interpretive program (the document is strictly a physical-predictions calculation with audit trail — all non-predictive content has been removed in the v0_42 release pass)

It is a working research document at the stage of external technical review.
